# NeuroReview AI 🚀
### Autonomous Workforce Intelligence Platform

NeuroReview AI is an advanced AI-powered workforce management web application that integrates **Generative AI** and **Agentic AI** to transform employee performance evaluation, organizational insights, and decision-making.

The platform provides role-based intelligent dashboards for Employees, Managers, HR, and Executives with autonomous AI agents that analyze performance data, generate insights, and simulate organizational outcomes.

---

## 🧠 Core Concept

Traditional performance systems are manual, biased, and reactive.

**NeuroReview AI** introduces:
- Generative AI → Creates insights, reviews, and development plans
- Agentic AI → Acts autonomously to monitor, analyze, and recommend actions

This creates a **self-improving intelligent organization system**.
